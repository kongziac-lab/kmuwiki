# Codex Skills

This folder backs up personal Codex skills so they can be reused on another computer.

## Install on Windows

Copy the skill folders in this directory into:

```powershell
C:\Users\<USER>\.codex\skills
```

For example:

```powershell
Copy-Item -Recurse .\codex-skills\badge-background-templates C:\Users\<USER>\.codex\skills\
```

Restart Codex or start a new thread after copying so the skill registry refreshes.

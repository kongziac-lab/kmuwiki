# 공문서 예시 모음

## 예시 1: 협조 요청 공문 (외부 발신)

```
계명대학교 국제처

수 신:  창춘대학교 국제교류처장
제 목:  2026학년도 파견 교원 명단 제출 요청

1. 계명대학교와 창춘대학교 간 교육 협력 협약(2023. 9. 1. 체결)에 의거함.

2. 2026학년도 1학기 파견 교원 관련 아래 사항을 요청함.
  가. 제출 서류
    1) 파견 교원 명단(성명, 담당 과목, 파견 기간)
    2) 교원별 이력서(중국어 또는 영문)
  나. 제출 기한: 2026. 4. 30.
  다. 제출 방법: 이메일 제출(international@kmu.ac.kr)

붙임  1. 파견 교원 명단 양식 1부.  끝.

                              계명대학교 국제처장
```

## 예시 2: 내부 기안문

```
계명대학교 국제처

수 신:  총장
제 목:  공자아카데미 운영 현황 및 2026년 사업 계획 보고

1. 공자아카데미 운영 현황(2025년 기준)은 다음과 같음.
  가. 수강생 현황
    1) 총 등록 수강생: 320명
    2) 전년 대비 증감: 15명 증가(4.9% 증가)
  나. 운영 프로그램
    1) 정규 중국어 강좌: 주 5회 운영
    2) 문화 체험 행사: 연 4회 실시
    3) HSK 시험 지원: 연 3회

2. 2026년 주요 사업 계획은 붙임과 같음.

붙임  1. 2026년 공자아카데미 사업 계획서 1부.
      2. 2025년 운영 실적 보고서 1부.  끝.

                              국제처장
```

## 예시 3: 시행문 (내부 공지)

```
계명대학교 국제처

수 신:  각 단과대학장
제 목:  외국인 유학생 등록금 장학금 신청 안내

1. 2026학년도 1학기 외국인 유학생 대상 등록금 장학금 신청을 다음과 같이 안내함.

2. 신청 자격: 직전 학기 성적 3.0 이상인 재학 중인 외국인 유학생

3. 신청 방법
  가. 신청 기간: 2026. 3. 1. ~ 2026. 3. 15.
  나. 제출 서류
    1) 장학금 신청서 1부
    2) 성적 증명서 1부
    3) 재학 증명서 1부
  다. 제출처: 국제처 유학생지원팀(성산관 308호)

4. 선발 기준
  가. 학업 성적: 70점
  나. 봉사 활동: 20점
  다. 어학 능력: 10점

5. 장학금 지급액: 금1,000,000원(일백만원) (등록금의 30% 이내)

붙임  1. 장학금 신청서 양식 1부.  끝.

                              계명대학교 국제처장
```
